# 🧪 Сортировка колб (Tube Sort Puzzle)

Логическая игра-головоломка на Python, где нужно отсортировать цветные предметы по колбам.

## 🎯 Правила игры
- У вас есть 6 колб: 4 заполнены предметами 4 разных цветов (A, B, C, D), 2 пустые
- В каждой колбе может быть не более 4 предметов
- Можно переливать **только верхний** предмет из колбы
- Предмет можно положить в колбу, если она пуста или верхний предмет такого же цвета
- Цель: собрать в каждой колбе 4 предмета одного цвета

## 🎮 Как играть
```bash
python tube_sort.py
Программа покажет текущее состояние колб и попросит вводить номера колб для переливания.

📋 Пример игрового процесса
text
=== Игра: Сортировка колб ===
1: A B C D
2: A B C D
3: A B C D
4: A B C D
5: [Пусто]
6: [Пусто]

Откуда перемещаем (1-6)? 1
Куда перемещаем (1-6)? 5
🧠 Особенности реализации
Объектно-ориентированный дизайн (классы Tube и Game)

Полная проверка правил переливания

Случайная генерация начальной позиции

Подсчёт количества ходов

Защита от некорректного ввода
